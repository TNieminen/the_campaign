Mundane Treasure
================

Mundane treasure and valuable objects. Roll on the table below (d12 twice, or
run `tools/loot.py`).

Gold values are tuned for a party of 4. Scale coin and object values by roughly
`party_size / 4` for larger or smaller groups.
